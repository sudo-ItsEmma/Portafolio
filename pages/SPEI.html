TuSPEI

Transferencias digitales accesibles para adultos mayores

Aplicación móvil diseñada para reducir la brecha digital financiera, priorizando simplicidad, accesibilidad multimodal y seguridad reforzada.

La Problemática

En México, el uso de banca digital por adultos mayores es reducido. Interfaces complejas, exceso de funciones y temor a cometer errores irreversibles en transferencias SPEI generan desconfianza y exclusión financiera.

El reto principal no es tecnológico, sino de confianza y usabilidad.

Solución Propuesta

TuSPEI minimiza el riesgo de error mediante verificación constante, confirmaciones visuales y auditivas, uso de biometría y autocompletado inteligente mediante enlaces o códigos QR.

Características Clave

Accesibilidad Multimodal

Comandos de voz, lectura en voz alta, botones grandes y alto contraste.

Verificación Inteligente

Resumen claro antes de cada transferencia para evitar errores irreversibles.

Seguridad Reforzada

Autenticación biométrica (huella / rostro) y PIN personal.

Arquitectura del Sistema

Frontend

Aplicación móvil nativa desarrollada con React Native, integrando reconocimiento de voz y biometría del dispositivo.

Backend (BFF)

Node.js con Express bajo patrón Backend for Frontend, actuando como intermediario seguro entre app y bancos.

Seguridad

OAuth 2.0, APIs de Open Banking y uso exclusivo de biometría nativa.

Persistencia

Base de datos PostgreSQL para almacenar únicamente datos no sensibles.
