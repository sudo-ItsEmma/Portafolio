SWAL

Sistema Web de Asesoramiento de Lectoescritura.

Innovación educativa para habilidades fundamentales.

Objetivo General

Desarrollar, mediante una metodología ágil, un sistema web de asesoramiento de lectoescritura para universitarios que provea herramientas para la práctica, retroalimentación y seguimiento de sus competencias.

El Desafío

Existía una brecha crítica en la gestión de procesos académicos relacionados con la lectoescritura. Era necesario desarrollar una solución digital eficiente que permitiera diferenciar claramente los flujos de trabajo según el rol del usuario.

Solución Propuesta

Se desarrolló una plataforma integral con navegación diferenciada: modelo HUB-AND-SPOKE para administradores y menú persistente para asesores y alumnos, optimizando eficiencia y claridad.

Stack Tecnológico

Frontend

HTML5, CSS3, JavaScript, Bulma

Backend

PHP, Arquitectura MVC

Base de Datos

MySQL

Diferenciador

Las plataformas educativas genéricas (como Google Classroom) se centran en la entrega de tareas, pero carecen de una visión macro del progreso institucional. Los administradores suelen perderse en "islas de datos" por cada aula virtual.

Desarrollé un motor de Analítica Centralizada que permite a la institución dejar de solo "gestionar tareas" para pasar a "tomar decisiones". Mientras otras plataformas muestran qué entregó el alumno, SWAL muestra cómo está evolucionando el nivel académico general mediante:

Mi Diferenciador Estratégico

Auditoria de Desempeño

Reportes de actividad del asesor y uso del sistema

Diagnóstico de Aprendizaje

Identificación de áreas de oportunidad y avance por nivel

Intelifencia Operativa

Filtros globales para reportes de cumplimiento inmediato y gráficas
